üöÄ Setup Guide

1

Get API Keys

Obtain API keys from all three sources:

# OpenStates: https://openstates.org/accounts/register/
# Congress.gov: https://api.congress.gov/sign-up/
# Google Civic: https://console.cloud.google.com/
2

Configure Environment

Create a .env file with your credentials:

DISCORD_TOKEN=your_bot_token
OPENSTATES_API_KEY=your_key
CONGRESS_GOV_API_KEY=your_key
GOOGLE_CIVIC_API_KEY=your_key
3

Install Dependencies

Set up Python virtual environment and install packages:

python -m venv venv
venv\Scripts\activate # Windows
pip install -r requirements.txt
4

Run the Bot

Start the bot:

python bot.py
5

Server Setup

Use /setup in your server to automatically create all roles, channels, and forums.
