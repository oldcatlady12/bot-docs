March Madness Bot

Overview

The March Madness Bot is a comprehensive NCAA Tournament bracket management and betting system. Create brackets, place bets on games, earn coins, and compete on leaderboards during March Madness.

🏀

Bracket Builder

Create and manage tournament brackets

💰

Betting System

Place bets on individual games

💸

Economy

Earn and spend coins throughout the tournament

🏅

Leaderboards

Compete against other users

Core Features

🏀 Bracket Management

Users can create tournament brackets with their predictions. The advanced bracket builder supports selecting winners for each round from First Four through the Championship game.

💰 Betting System

Place bets on upcoming tournament games. Supports moneylines, spreads, and more. Payouts calculated based on odds from data providers.

💸 Economy System

Users start with coins and can earn more through:

  • Daily claims
  • Correct bracket picks
  • Winning bets
  • Bonus activities

📈 Bracket Visualization

Generate visual bracket images showing all picks and results. Track your progress through each round.

Database Schema

The bot uses SQLite with these main tables:

-- Brackets table brackets: bracket_id, user_id, guild_id, bracket_data, elimination_picks, score, created_at -- Bets table bets: bet_id, user_id, guild_id, game_id, bet_type, amount, odds, status -- Economy table economy: user_id, guild_id, balance, total_earned, total_wagered, last_daily -- Game results (tournament games) game_results: game_id, round, winner, score, settled

Management Tools

The manage.py script provides admin utilities:

python manage.py # Available operations: # - Backup database # - Export brackets to JSON # - Show statistics # - Reset economy (with confirmation) # - Reset tournament (with confirmation)

Seasonal Usage

This bot is primarily active during the NCAA Tournament (March-April). The workflow is:

  1. Before tournament: Users create brackets
  2. Tournament starts: Brackets lock, betting opens
  3. Each round: Scores update, bets settle
  4. End: Final leaderboards, economy persists
  5. Off-season: Use manage.py to reset for next year
