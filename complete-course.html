Module 1: Python

Async/Await

Handle operations that take time without blocking.

Why Async?

Discord bots need to handle multiple users at once. Async lets your bot respond to new messages while waiting for API calls.

async_example.py Python
import asyncio # Async function async def fetch_data(): print("Fetching...") await asyncio.sleep(2) # Wait 2 seconds return "Data!" # Call async function async def main(): result = await fetch_data() print(result) # Run it asyncio.run(main())

Key Rules

  • Use async def to define an async function
  • Use await when calling another async function
  • You can only use await inside an async function
