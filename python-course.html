Lesson 2

Variables & Data Types

Learn how to store and work with different types of data in Python.

üì¶ What are Variables?

Variables are like labeled containers that store data. You give them a name, and Python remembers the value you put in them.

variables.py Python 
# Creating variables - no special keyword needed!
name = "Arthur"           # String (text)
age = 25                   # Integer (whole number)
score = 99.5               # Float (decimal number)
is_admin = True            # Boolean (True or False)

# Using variables
print("Hello, " + name)     # Output: Hello, Arthur
print("Age:", age)          # Output: Age: 25
Variable Assignment
name

"Arthur"

The variable name now holds the value "Arthur"

üìä Python Data Types

Python has several built-in data types. Here are the most common ones:

Type Description Example
str String - text in quotes "Hello" or 'Hello'
int Integer - whole numbers 42, -10, 0
float Float - decimal numbers 3.14, -0.5
bool Boolean - True or False True, False
list Ordered collection [1, 2, 3]
dict Key-value pairs {"name": "Arthur"}

üîç Checking Types

Use the type() function to check what type a variable is:

‚öñÔ∏è Comparison Operators

Comparison operators compare two values and return True or False:

Operator Meaning Example Result
== Equal to 5 == 5 True

> Greater than 5 > 3 True
< Less than 5 < 3 False
>= Greater or equal 5 >= 5 True

‚ö†Ô∏è Common Mistake: = vs ==
  • = is for assignment (setting a value)
  • == is for comparison (checking equality)
x = 5      # Assignment: x now equals 5
x == 5     # Comparison: is x equal to 5? (True)

üîó Logical Operators

Combine multiple conditions:

logical.py Python 
age = 25
is_member = True

# AND - both must be true
can_vote = age >= 18 and is_member    # True

# OR - at least one must be true
gets_discount = age < 18 or age > 65  # False

# NOT - flips True to False and vice versa
is_guest = not is_member                # False

Lesson 4

If Statements (Conditionals)

Learn how to make decisions in your code based on conditions.

üîÄ Making Decisions

If statements let your program make decisions. They check a condition and run different code based on whether it's True or False.

if_example.py Python 
score = 85

if score >= 90:
    print("A - Excellent!")
elif score >= 80:
    print("B - Good job!")
elif score >= 70:
    print("C - Not bad")
else:
    print("Need improvement")

# Output: B - Good job!
üîë Key Points
  • if - First condition to check
  • elif - Additional conditions (short for "else if")
  • else - What to do if no conditions are True
  • Indentation matters! Code inside the if must be indented with 4 spaces
