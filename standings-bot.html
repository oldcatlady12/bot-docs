Features

Automatic Standings Channel

When you run /setup channel, the bot creates a dedicated channel called #sports-standings that displays current standings for all enabled leagues. The message is automatically updated every 30 minutes.

Forum Integration

Link a sports forum channel with /setup forum and the bot will automatically post standings updates to new forum threads.

Premium Features

  • Custom Update Frequency: Set standings updates from 1-7 days
  • Priority Support: Get help faster with premium support
  • Coming Soon: Custom league filtering, team highlights, and more

Pricing: $4.99/month or $49.99/year
