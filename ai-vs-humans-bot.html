üíª Commands

Command Description Access
/play Start a new game round. Optionally specify mode (image, text, art, music, random). Everyone
/quick Instantly start a random mode game without options. Everyone
/stats View your personal statistics, accuracy rates, and ranking. Everyone
/leaderboard View the server or global leaderboard by ELO rating. Everyone
/challenge @user Challenge another user to a head-to-head match. Everyone
/tournament Start a multi-round tournament for multiple players. Everyone
/daily Play the daily challenge for bonus points. Everyone
/streak View your current winning streak and best streak. Everyone
/learn Access educational content about AI detection techniques. Everyone
/add-content Submit new AI or human content to the game database. Admin

üöÄ Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot.

2

Configure Environment

Create a .env file with your bot token:

DISCORD_TOKEN=your_bot_token_here
DATABASE_PATH=data/game.db
3

Install Dependencies

Set up Python virtual environment and install packages:

python -m venv venv
venv\Scripts\activate # Windows
pip install -r requirements.txt
4

Run the Bot

Start the bot:

python main.py
