Changelog Bot

Overview

The Changelog Bot monitors CHANGELOG.md files across all your bots and automatically posts update announcements to Discord when new entries are detected.

Changelog Format

CHANGELOG.md files should follow this format for proper parsing:

## [1.2.0] - 2025-01-06 ### Added - New feature description ### Changed - Modified behavior description ### Fixed - Bug fix description ---

Each entry starts with ## and ends with --- (or the next entry).

Setup

Installation

cd changelog-bot python -m venv venv .\venv\Scripts\activate pip install -r requirements.txt # Configure .env # DISCORD_TOKEN=your_token python bot.py

Adding New Bots to Track

Edit the BOTS_CONFIG dictionary in bot.py:

BOTS_CONFIG = { 'your_bot': { 'name': 'Your Bot Name', 'emoji': '🌟', 'color': 0x3498db, 'changelog_path': r'c:\path\to\CHANGELOG.md' } }

How Detection Works

  1. Bot checks CHANGELOG.md files periodically
  2. Parses the most recent entry (first ## section)
  3. Compares entry date to last posted date
  4. If new, posts formatted embed to configured channel
  5. Updates last posted date to prevent duplicates
