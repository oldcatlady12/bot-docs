Invite Battles Bot

King of the Hill for Server Invites

A competitive Discord game where users battle for their server invite to be displayed in a channel. Submit an invite, set a secret number, and defend your throne while challengers try to guess it. Features perks, leaderboards, and multi-server support.

⚔ How It Works

Submit Invite

Post a discord.gg link or use /invite in a game channel

🔒

Set Secret Number

Pick a number 1-100 via a private modal popup

🎯

Others use /guess — within 10 of the number wins

👑

New King

Winner's invite replaces the old one on the hill

🔄 Game Flow

1

User submits an invite link

Via /invite <link> or by posting a discord.gg link in a game channel.

2

If the hill is empty — become the King

A modal appears asking for a secret number (1-100). The invite embed is posted to the channel.

3

If the hill is occupied — join the queue

Your invite is queued. Use /guess to try to dethrone the current king.

4

Guess within 10 of the secret number to win

Each user gets one guess per day per channel. If |guess - secret| ≤ 10, you win!

5

Winner takes the hill

The old invite is deleted, the reign is archived, and the winner sets their own secret number.

6

Auto-expiration after 7 days

If nobody guesses the number within a week, the invite expires and the next person in queue is promoted.

✨ Key Features

👑 King of the Hill

Only one invite can reign per channel at a time. Defend your throne by choosing a hard-to-guess number.

🔎 Dual Input Detection

Submit invites via slash command /invite or just paste a discord.gg link — the bot detects both.

📅 Daily Guess Limit

Each user gets one guess per day per channel, keeping the game strategic and drawn out.

⏰ 7-Day Expiration

Unguessed invites automatically expire after one week. The queue advances automatically.

📊 Interactive Leaderboards

Dropdown menus to switch views (longest streak, most wins, top defenders) with pagination buttons.

🌐 Multi-Server Support

Independent game state per server and channel. Global leaderboard aggregates across all servers.

💻 Slash Commands

Command Description Access
/invite <link> Submit a Discord invite link. If the hill is empty, you become king. Otherwise, your invite is queued. Everyone
/guess <number> Guess the king's secret number (1-100). Within 10 wins. One guess per day. Everyone
/tip Get a hint about the secret number (20-wide range). Requires Guess Tip perk role. Perk Role
/leaderboard View interactive leaderboard with dropdown views and pagination (server + global). Everyone
/setup Initialize Invite Battles for the server. Run once to get started. Admin
/config channel add/remove Add or remove a text channel as a game channel. Admin
/config channel list List all configured game channels. Admin
/config perks add/remove Assign or remove a perk (Extra Life, Number Reset, Guess Tip) to a role. Admin
/config perks list List all perk role assignments. Admin
/config tips <limit> Set the daily tip limit per user (0-10). Admin

🌟 Special Perks

Admins assign perks to roles via /config perks add <perk> @role. Members with the role gain the perk ability.

💛 Extra Life

If the king has this perk and someone guesses their number, instead of losing the hill, the bot automatically picks a new random secret number. One-time use per reign.

🔄 Number Reset

Allows resetting the king's secret number to a new random value. Useful for refreshing a stale game when the number hasn't been guessed.

💡 Guess Tip

Use /tip to get a hint — a 20-wide range containing the secret number (e.g., "between 35 and 55"). Limited uses per day, configurable by admin.

🏆 Leaderboard System

The /leaderboard command displays an interactive embed with a dropdown menu and pagination buttons.

🕐 Longest Streak (Server)

Users whose invites lasted the longest on the hill, measured by duration.

🏆 Most Wins (Server)

Users who have dethroned the most kings by guessing correctly.

🛡 Top Defenders (Server)

Users who survived the most failed guesses during a single reign.

👑 Current King

Shows the active invite holder in each game channel.

🌐 Global Longest Streak

Longest reigns across all servers the bot is in.

🌐 Global Most Wins

Top dethronements across all servers.

🗃 Database Architecture

SQLite database with 8 tables, all scoped by guild_id for multi-server isolation.

guild_config

Per-server settings: setup status, daily tip limit.

game_channels

Which channels run the game. Multiple per server.

active_invites

Current king per channel: user, link, secret number, expiration, guess count.

invite_queue

FIFO queue of pending invites waiting to claim the hill.

guess_log

Every guess recorded. Enforces one guess per user per day.

reign_history

Archived reigns with duration, guesses survived, and who dethroned them. Powers leaderboards.

perk_roles

Maps Discord roles to perks (extra_life, number_reset, guess_tip).

tip_usage

Tracks daily tip usage per user to enforce configurable limits.

🚀 Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot. Enable Message Content and Server Members privileged intents.

2

Configure Bot Token

Edit config.json with your credentials:

{
  "token": "YOUR_BOT_TOKEN",
  "clientId": "YOUR_APPLICATION_ID",
  "guildId": "YOUR_DEV_GUILD_ID"
}
3

Install Dependencies

npm install
4

Register Slash Commands

npm run deploy        # Global (all servers)
npm run deploy -- --dev  # Dev guild only (instant)
5

Start the Bot

npm start
6

In Discord: Initialize a Server

/setup
/config channel add #invite-battles
