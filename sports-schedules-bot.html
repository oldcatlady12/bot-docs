Sports Schedules Bot

Overview

The Sports Schedules Bot posts daily game schedules with links to game posts created by the Sports Bot. It automatically updates schedules every 10 minutes and offers premium DM notifications for game alerts.

Integration with Sports Bot

This bot is designed to work alongside the Sports Bot. It:

  • Finds game posts created by Sports Bot in forum channels
  • Adds direct links to those game threads in schedule messages
  • Uses team roles to notify premium users when their team's game post is created
  • Fetches live game data from ESPN API

Automatic Features

Daily Schedule Posts

At a configurable time each day (default: 7:00 AM ET), the bot automatically posts the day's game schedule to all configured channels.

Auto-Updating Messages

Schedule messages are automatically updated every 10 minutes to reflect:

  • New game posts created by Sports Bot
  • Game status changes (upcoming, in progress, final)
  • Score updates for live games

DM Notifications (Premium)

Premium users receive direct message notifications when:

  • A game post is created for a team they have a role for
  • Notifications include direct links to the game thread
  • Notifications are checked every few minutes

Premium Features

What Premium Users Get:

  • DM Notifications: Get alerted when game posts are created for your teams
  • Team Role Integration: Notifications based on team roles you have in the server
  • Priority Updates: Be the first to know when game threads are available

Default Duration: 30 days per grant

Quick Setup

# 1. Set the schedule channel !schedule_set_channel #daily-schedules # 2. Post a schedule immediately !schedule_post # 3. Grant premium to a user !schedule_grant_premium @user 30 # 4. Check premium subscribers !schedule_list_premium
