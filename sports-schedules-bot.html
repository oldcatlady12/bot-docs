Sports Schedules Bot

Overview

The Sports Schedules Bot posts game schedules with direct links to game posts created by the Sports Bot. It auto-updates when game posts are created and includes a Game Results dropdown that connects to the AI vs Humans Bot for viewing prediction outcomes.

Bot Integration

Sports Bot Integration

Works alongside the Sports Bot to:

  • Monitor when game posts are created in forum channels
  • Auto-update schedule messages with channel mentions
  • Remove games from schedule when they finish
  • Detect changes by watching the sports-bot settings file

AI vs Humans Bot Integration NEW

Connects to the AI vs Humans Bot database to display game results:

  • Results Dropdown: Select completed games from a dropdown menu
  • Final Scores: View game outcomes with winner indicators
  • AI Predictions: See what each AI model predicted and accuracy
  • Human Stats: View how many users predicted correctly
  • AI vs Humans: See who won the prediction battle for each game

Game Results Feature NEW

How It Works

The results message displays a dropdown menu with completed games from the AI vs Humans Bot:

  • Step 1: Select a completed game from the dropdown
  • Step 2: View an ephemeral embed with full results
  • Step 3: See final score, AI predictions, and human stats
  • Step 4: Find out who won: AI or Humans!

Results Display Includes:

Game outcome with winner indicator and total points vs O/U line

🤖

👥

Human Stats

Total predictions, pick distribution, and accuracy rates

⚔

Battle Results

Who won the game: AI or Humans with accuracy comparison

Auto-Update System

Settings File Watcher

The bot monitors the Sports Bot's settings file for changes:

  • Check Interval: Every 3 seconds
  • Detection: Watches file modification time
  • Action: Immediately updates schedule when changes detected

Schedule Updater

When games are active, the schedule auto-updates:

  • Interval: Every 10 minutes
  • Triggers: New game posts, game status changes
  • Auto-Stop: Stops updating when no active games
