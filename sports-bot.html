Sports Bot

Overview

The Sports Bot provides real-time sports updates, live game tracking, injury reports, and game statistics. It supports multiple sports leagues and creates live game threads in Discord forum channels.

Real-time score updates with Components V2

Commands

All commands require Administrator permissions.

Setup Commands Admin Only

CommandDescription
!setup_sports <sports>Enable sports tracking for the guild. Accepts multiple sport names (e.g., !setup_sports nba nfl mlb)
!set_category [category_id]Set the category where game forums will be created. Creates a new "Live Games" category if no ID provided.
!setup_alerts <sports>Create alert roles for sports leagues. Use all for all sports (e.g., !setup_alerts nba nfl or !setup_alerts all)
!delete_alerts <sports>Delete alert roles for specified sports or all sports
!setup_guideCreate the user guide channel in the same category as game posts
!set_admin_guide <channel_id>Post complete admin documentation to a specified channel

Team Tracking Commands Admin Only

CommandDescription
!track_team <sport> <team_name>Track a specific team's games for a particular sport
!set_team_role @Role <TeamName>Link a Discord role to a team for game alerts (e.g., !set_team_role @Lakers Los Angeles Lakers). Team roles get mentioned instead of general sport role.
!remove_team_role <TeamName>Remove a team role link
!list_team_rolesList all team-to-role mappings for this server

Monitoring Commands Admin Only

CommandDescription
!bot_statusCheck bot's current status including enabled sports, live games, upcoming games, finished games, and monitoring status
!restart_gamesDelete all existing game posts and recreate them from scratch for games in progress or starting within 30 minutes
!check_gamesManually check for games and create posts for any missing. Resumes monitoring if paused.

Game Thread Features

When a game starts, the bot automatically creates a forum thread with:

  • Live score updates every 30 seconds
  • Quarter/Half/Period transitions with stats
  • Lead changes and key plays
  • Final score with complete box score
  • Components V2 styling for modern Discord look
  • Links to both teams' ESPN schedule pages in the game description
  • Channel mentions for other live games (not URL links)
  • Thread title automatically updates to "FINAL" with scores when game ends
