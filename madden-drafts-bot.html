Franchise Draft Scouting & Predictions

🏈 Madden Draft Scouting System

Complete draft scouting and prediction system for Madden franchise modes. View detailed scouting reports, track Madden's rankings, make predictions, and manage multiple leagues and seasons.

Python 3.10+ discord.py 2.0+ JSON Database Slash Commands

🛠 Key Features

🔍

Player Scouting

Detailed scouting reports with strengths, weaknesses, development traits, archetypes, and NFL comparisons.

🏆

Rankings System

View Madden's official rankings alongside custom potential rankings for each position group.

📋

Draft Predictions

Users can predict where players will be drafted. Predictions are tracked and scored for accuracy.

🌐

Multi-League Support

Track multiple leagues with multiple seasons each. Perfect for franchise communities with several groups.

📷

Player Images

Admins can add player portrait images to enhance scouting reports and the draft guide website.

🔗

Push to Discord

Admin can edit players via local website and push updates to Discord for community announcements.

💻 User Commands

/lookup User

Look up a player's full scouting report including development, archetype, strengths, weaknesses, and draft status.

Parameters
player_name - The name of the player to look up
/potential User

View potential rankings for a position group based on scouting analysis.

Parameters
position - Position group (QB, HB, WR, TE, OL, DL, LB, CB, S)
/rankings User

View Madden's official draft rankings for a position or overall.

Parameters
position - Position group or "ALL" for overall rankings
/predict User

Make a draft prediction for where a player will be selected.

Parameters
player_name - The player's name
pick_number - Predicted overall pick (1-256)
team - Predicted team to draft the player
/mypredictions User

View all of your draft predictions and their results.

/leagueinfo User

View current league information including season and all available leagues.

🔐 Admin Commands

/addplayer Admin

Add a new player to the current draft class.

Parameters
name - Player's full name
position - Player's position
college - Player's college
age - Player's age
development - Development trait (Normal, Star, Superstar, X-Factor)
archetype - Player archetype (optional)
/setimage Admin

Set a player's portrait image URL.

Parameters
player_name - The player's name
image_url - URL to the player's image
/setscouting Admin

Set or update a player's scouting report.

Parameters
player_name - The player's name
report - The scouting report text
strengths - Comma-separated list of strengths
weaknesses - Comma-separated list of weaknesses
/setdrafted Admin

Mark a player as drafted and update prediction results.

Parameters
player_name - The player's name
team - Team that drafted the player
round_num - Draft round (1-7)
pick_num - Overall pick number
/setrank Admin

Set a player's Madden rank and/or potential rank.

Parameters
player_name - The player's name
madden_rank - Madden's official rank
potential_rank - Custom potential rank
/newseason Admin

Start a new season in the current league.

/newleague Admin

Create a new league and set it as active.

Parameters
name - Name for the new league
/setleague Admin

Switch to a different league.

Parameters
league_number - The league number to switch to

📂 Data Structure

The bot uses JSON files for data storage, located in the data/ directory:

players.json

Contains all player data including name, position, college, development trait, scouting report, rankings, and draft results.

leagues.json

Tracks all leagues, their current seasons, and which league is currently active.

predictions.json

Stores all user predictions with their results (pending, correct, wrong).
