Core Features

🏈 EA Madden Integration

Login to EA website with encrypted credentials, fetch franchise data including teams, players, stats, and game results. Supports Madden 23, 24, 25 with auto-detection.

🎯 Daily Fantasy (DFS)

Create salary-capped lineups each game week. Features include:

  • $50,000 salary cap per lineup
  • 8 roster slots: QB, RB1, RB2, WR1, WR2, WR3, TE, FLEX
  • Entry fees from 10 to 10,000 coins
  • Automatic prize pool distribution

🏆 Season-Long Fantasy

Traditional fantasy leagues with drafts, weekly matchups, trades, and waivers. Supports multiple scoring formats:

  • Standard: 4pt passing TD, 6pt rush/rec TD, no PPR
  • PPR: 1 point per reception
  • Half PPR: 0.5 points per reception
  • Madden: Custom Madden-specific scoring with bonuses

💰 Betting System

Bet on Madden franchise games with spreads, moneylines, and over/under totals. Lines are auto-generated based on team ratings.
