✨ Key Features

🕑 Auto-Close Predictions

Predictions automatically lock when games start. No late picks allowed!

🔄 Daily Refresh

All game messages update at 8 AM ET daily with fresh matchups.

📊 Detailed Stats

Track your accuracy by sport, pick type (winner/O-U), and time period.

💰 Economy Integration

💻 Commands

/ai-records View individual AI model performance records. Everyone
/setup-sport Initialize a sport with forum channel and auto-posts. Admin

/generate-ai-predictions Generate AI predictions for upcoming games. Admin

❓ How It Works

1

Select a Game

Use the dropdown in the Daily Games post to select a matchup you want to predict.

2

Make Your Picks

Choose the winner (Home/Away) and Over/Under on the total points line.

3

Wait for Results

Predictions lock at game time. Results post automatically when games finish.

4

Track Your Record

Check the leaderboard to see how you stack up against other humans AND the AI!

🚀 Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot. Enable Server Members Intent and Message Content Intent.

2

Configure Environment

Create a .env file with your tokens:

DISCORD_BOT_TOKEN=your_discord_bot_token
OPENAI_API_KEY=your_openai_key (optional)
ANTHROPIC_API_KEY=your_anthropic_key (optional)
ECONOMY_BOT_DB_PATH=path/to/economy.db (optional)
3

Install Dependencies

Set up Python and install packages:

pip install -r requirements.txt
4

Run the Bot

Start the bot:

python main.py
5

Setup Sports

Use /setup-sport in a forum channel to initialize each sport you want to track.
