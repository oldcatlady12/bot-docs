❓ How It Works

1

Select a Game

Use the dropdown in the Human Predictions thread to select a matchup you want to predict.

2

Make Your Picks

Choose the winner (Home/Away) and Over/Under on the total points line. Optionally place a bet with economy coins.

3

AI Makes Its Picks Too

Three AI models independently analyze each game and make their own winner + O/U predictions with detailed reasoning.

4

Wait for Results

Predictions lock at game time. Final scores are fetched from ESPN automatically.

5

Get Results & Payouts

Predictions are graded, stats updated, bet payouts processed, and you get a DM with your results.

6

Track the War

Check the leaderboard to see how humans stack up against the AI. Use /war-stats for the full breakdown!

🚀 Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot. Enable Server Members Intent and Message Content Intent.

2

Configure Environment

Create a .env file with your tokens:

DISCORD_BOT_TOKEN=your_discord_bot_token
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key
ECONOMY_BOT_DB_PATH=path/to/economy.db (optional)
3

Install Dependencies

Set up Python 3.10+ and install packages:

pip install -r requirements.txt
4

Run the Bot

python main.py
5

Setup Sports

Use /setup-sport in a forum channel to initialize each sport. Then use /setup-results-forum for a results forum.

6

Fetch Games & Post Predictions

Use /fetch-games to pull today's games, then /post-ai-predictions to generate and post AI picks. Users can then make their predictions via the dropdown in the forum thread.

7

NFL Season Predictions (Optional)

Use /setup-season 2025 to create season prediction threads. Then /generate-season-predictions 2025 for AI picks. Lock with /lock-season when the NFL season begins.
