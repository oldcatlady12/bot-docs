üíª Slash Commands

Command Description Access
/balance Check your or another user's balance in the current server. Everyone
/bank View your bank overview with all accounts across servers. Everyone
/transfer Send currency to another user in the same server. Everyone
/exchange Exchange currency between different servers or to BotCoin. Everyone
/leaderboard View various leaderboards (balance, global, transfers, etc). Everyone
/lottery View lottery information and your tickets. Everyone
/shop browse Browse available items in the server shop. Everyone
/shop inventory View your purchased items. Everyone
/admin currency Set server currency name and emoji. Admin
/admin rates Configure earning rates for messages/voice/reactions. Admin
/admin exchange panel Create an exchange rate panel in a channel. Admin
/admin lottery panel Create a lottery panel in a channel. Admin
/admin shop add Add a new item to the shop. Admin

üéØ Activity Rewards & Lottery Tickets

Users earn currency and lottery tickets through server activity:

üí¨ Messages

Earn coins per message (with cooldown). Also earns lottery tickets based on message count.

üòÄ Reactions

Earn coins for adding reactions. Every 5 reactions earns a lottery ticket.

üé§ Voice Activity

Earn coins per minute in voice channels (must be unmuted with others). Also earns lottery tickets.

üé´ Lottery Tickets

Tickets are automatically entered into Daily, Weekly, and Monthly lottery drawings.

üé∞ Lottery System

Three lottery types with automatic drawings and tiered prizes:

üåÖ Daily Lottery

Drawn at midnight UTC daily. 1st, 2nd, and 3rd place prizes in BotCoin.

üìÖ Weekly Lottery

Drawn Sunday midnight UTC. Larger prizes for weekly participation.

üóìÔ∏è Monthly Lottery

Drawn 1st of each month. Biggest jackpots for dedicated participants.

üìä Lottery Panel

Admin-created panel showing all lotteries, ticket counts, recent winners, and how to earn tickets.

üöÄ Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot.

2

Configure Environment

Create a .env file with your credentials:

DISCORD_TOKEN=your_bot_token_here
CLIENT_ID=your_client_id
DATABASE_URL="file:./dev.db"
3

Install & Setup Database

Install dependencies and initialize the database:

npm install
npx prisma db push
4

Run the Bot

Start the bot:

npx ts-node src/index.ts
# Or for development with auto-reload:
npm run dev

Use /admin rates to configure how much users earn from messages, voice, and reactions.
