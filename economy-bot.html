üéØ Activity Rewards & Lottery Tickets

Users earn currency and lottery tickets through server activity:

üí¨ Messages

Earn coins per message (with cooldown). Also earns lottery tickets based on message count.

üòÄ Reactions

Earn coins for adding reactions. Every 5 reactions earns a lottery ticket.

üé§ Voice Activity

Earn coins per minute in voice channels (must be unmuted with others). Also earns lottery tickets.

üé´ Lottery Tickets

Tickets are automatically entered into Daily, Weekly, and Monthly lottery drawings.

üé∞ Lottery System

Three lottery types with automatic drawings and tiered prizes:

üåÖ Daily Lottery

Drawn at midnight UTC daily. 1st, 2nd, and 3rd place prizes in BotCoin.

üìÖ Weekly Lottery

Drawn Sunday midnight UTC. Larger prizes for weekly participation.

üóìÔ∏è Monthly Lottery

Drawn 1st of each month. Biggest jackpots for dedicated participants.

üìä Lottery Panel

Admin-created panel showing all lotteries, ticket counts, recent winners, and how to earn tickets.

üí∞ Tax System NEW

Automated weekly tax collection with a communal pot for server projects:

üìÖ Weekly Collection

Taxes are automatically collected every Sunday at midnight UTC. Configurable tax rate (default 10%).

üíµ Tax Pot

All collected taxes go into a communal pot. View balance with /tax pot.

‚öôÔ∏è Admin Control

Enable/disable taxes, set rates, and distribute funds using admin commands.

üìä Transparency

Tax summaries posted to economy alert channel showing total collected and users taxed.

üí≥ Loan System NEW

Borrow currency with automatic repayments and bankruptcy protection:

üìù Request Loans

Use /loan request to borrow currency. Loan limits based on management tier (1,000 - 50,000).

üí∏ Auto-Repayment

Loans are automatically repaid when your balance reaches the loan amount. Processed every 10 minutes.

üìâ Interest Rates

Server-configurable interest rates. Admins can start polls to change rates via voting.

üè¶ Bankruptcy

Use /loan bankruptcy to clear all debts. After 2 bankruptcies, you're permanently banned from loans.

Loan Limits by Management Tier

Tier 1: Up to 1,000 currency
Tier 2: Up to 5,000 currency
Tier 3: Up to 10,000 currency
Tier 4: Up to 25,000 currency
Tier 5: Up to 50,000 currency

üìà Investment System NEW

üåê Cross-Server Investing

Use BotCoin to buy ownership shares in other servers. Prices fluctuate based on server activity.

üíµ Daily Dividends

Automatically receive dividends at 6 AM UTC based on your ownership percentage and server revenue.

üìä Investment Panel

Admin-created panel showing available servers, prices, and your current holdings.

üíº Portfolio View

Use /invest portfolio to see all your investments, earnings, and performance.

üó≥Ô∏è Voting System NEW

Democratic spending of the tax pot through community voting:

üìã Propose Spending

Management roles can propose using tax pot funds for grants, projects, or events.

‚úÖ Cast Votes

Members with voting privileges can vote FOR or AGAINST proposals.

üìä Majority Rules

Simple majority (50%+) passes. Proposals that would deplete the pot require 2/3 majority.

‚è∞ Auto-Processing

Expired votes are automatically tallied hourly. Passed proposals execute immediately.

‚ö†Ô∏è Warning & Fine System NEW

üìù Issue Warnings

üëÆ Fine Manager Role

üìú Warning History

üîó Link Servers

üí∞ Daily Tax Flow

üöÄ Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot.

2

Configure Environment

Create a .env file with your credentials:

DISCORD_TOKEN=your_bot_token_here
CLIENT_ID=your_client_id
DATABASE_URL="file:./dev.db"
3

Install & Setup Database

Install dependencies and initialize the database:

npm install
npx prisma db push
4

Run the Bot

Start the bot:

npx ts-node src/index.ts
# Or for development with auto-reload:
npm run dev
