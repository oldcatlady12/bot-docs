üöÄ Setup Guide

1

Create a Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot. Copy the bot token and client ID.

2

Get Google Places API Key

Create a project in Google Cloud Console, enable Places API and Geocoding API, then create an API key.

3

Configure Environment

Create a .env file with your credentials:

DISCORD_TOKEN=your_bot_token_here
CLIENT_ID=your_client_id_here
GOOGLE_PLACES_API_KEY=your_api_key_here
4

Install & Run

Install dependencies and start the bot:

npm install
npm start
5

Invite to Server

Use the OAuth2 URL Generator with bot and applications.commands scopes. Required permissions: Send Messages, Embed Links, Read Message History, Use Slash Commands.

6

Setup the Panel

Use /setup-testing-panel in your desired channel to create the interactive testing panel with buttons and dropdown menus.
