Discord Partner Bot

Server Partnership Management

A Discord bot for managing server partnerships, staff recruitment, job listings, and boost exchanges. Each category gets its own independent embed with Browse, Join, Edit, Leave, and Bump buttons. Features admin approval, activity tracking, DM notifications, and auto-refreshing embeds.

🔄 How It Works

For Users

1

Browse Listings

Each category has its own embed in the channel. Click Browse to see approved users in that category with a dropdown selector.

2

Join a Category

Click Join on a specific embed to apply. Fill out a description and optional invite link via a modal popup. Your application goes to admin review.

3

Get Approved

Admins review your application in the admin channel with Approve/Deny buttons. You'll receive a DM when your application is approved or denied.

4

Bump Your Listing

Click Bump to move your name to the top of the category list. Configurable cooldown (default 24 hours). Recent bumps are shown on each embed.

5

Edit or Leave

Click Edit to update your listing (requires re-approval). Click Leave to remove yourself from that category.

For Admins

1

Configure Channels

Set the partner channel for embeds and the admin channel for application reviews using slash commands.

2

Enable Categories

Enable/disable individual category embeds per server. Each enabled category gets its own embed with all buttons.

3

Review Applications

Approve or deny applications from the admin channel. Use /partner pending to see up to 5 pending applications at a time.

4

Monitor Activity

Add channels to track user activity. Users who have messaged in monitored channels within 30 days are marked as active.

✨ Key Features

📌 Per-Category Embeds

Each of the 4 categories gets its own embed message with independent Browse, Join, Edit, Leave, and Bump buttons.

✅ Admin Approval

All applications and edits go through admin review before appearing in the listings. Approve/Deny with interactive buttons.

📈 Bump System

Users can bump their listing to the top of the browse list. Configurable cooldown (1-168 hours). Recent bumps shown on each embed.

👥 Activity Tracking

Monitor specific channels for user messages. Activity status (active/inactive) shown in browse results based on 30-day window.

💬 DM Notifications

Users receive DMs when their application is approved or denied, keeping them informed without cluttering the server.

🔎 Browse with Details

Select users from a dropdown to view their description, invite link, Discord account age, join date, activity status, and other categories.

🌐 Server Invites

Partners can share invite links with clickable "Join Server" buttons shown in their browse details.

🔄 Auto Refresh

Embeds automatically refresh every hour and immediately when listings are approved or removed.

🚫 Auto Cleanup

When a member leaves the server, all their listings are automatically removed and embeds are refreshed.

📋 Multi-Category

Users can join multiple categories independently, each with its own description and invite link.

🛠 Admin Controls

Enable/disable specific embeds per server. Configure bump cooldown, monitor channels, and manage pending applications.

📡 Ephemeral Messages

All user interactions are ephemeral (only visible to the user), keeping the channel clean and clutter-free.

💻 Admin Commands

All commands use the /partner prefix and require Administrator permission. User interactions are button-based (Browse, Join, Edit, Leave, Bump).

Command Description
/partner set-channel #channel Set the channel where partnership embeds are posted
/partner set-admin-channel #channel Set the channel where admin application reviews are sent
/partner enable <category> Enable a specific category embed (Partnership, Staff, Work, Boosts)
/partner disable <category> Disable a category embed and remove it from the channel
/partner monitor-add #channel Add a channel to track user activity (messages in last 30 days)
/partner monitor-remove #channel Remove a channel from activity monitoring
/partner monitor-list List all currently monitored channels
/partner set-bump-cooldown <hours> Set the cooldown between bumps (1-168 hours, default: 24)
/partner pending View and manage up to 5 pending applications at a time
/partner status Show full bot configuration: channels, enabled embeds, listing counts
/partner refresh Manually refresh all enabled partnership embeds

🕹 Embed Buttons (User Actions)

🔎 Browse

View all approved users in the category. Select a user from the dropdown to see their full details, invite link, and DM mention.

📈 Bump

Move your name to the top of the browse list. Subject to a cooldown timer. Shows next available bump time if on cooldown.

➕ Join

Apply to join the category. Opens a modal to enter your description and optional invite link. Sent for admin approval.

✏ Edit

Update your existing listing. Opens a pre-filled modal. Changes require admin re-approval before going live.

🚪 Leave

Remove yourself from the category. Your listing is deleted immediately and the embed is refreshed.

🗃 Database Architecture

SQLite database with 5 tables, all scoped by guild_id for multi-server isolation. Uses WAL journal mode for performance.

guild_settings

Per-server configuration: partner channel, admin channel, bump cooldown hours.

guild_embeds

Tracks which category embeds are enabled per server and their Discord message IDs for editing.

partners

All partner listings with category, description, invite link, status (pending/approved/denied), and bump timestamp.

monitored_channels

Channels tracked for user activity. Messages in these channels update the user's activity timestamp.

user_activity

Tracks last message timestamp per user per guild. Used to determine active/inactive status (30-day window).

🚀 Setup Guide

1

Create Discord Bot

Go to the Discord Developer Portal, create a new application, and add a bot. Enable Server Members Intent under Privileged Gateway Intents.

2

Configure Environment

Copy .env.example to .env and add your bot token and client ID:

copy .env.example .env
3

Install Dependencies

npm install
4

Deploy Slash Commands

npm run deploy
5

Start the Bot

npm start
6

In Discord: Configure the Bot

/partner set-admin-channel #admin-reviews
/partner set-channel #partnerships
/partner enable Partnership
/partner enable Looking for Staff
/partner enable Looking for Work
/partner enable Looking for Boosts
/partner monitor-add #general

Set up admin and partner channels, enable desired categories, and add channels for activity monitoring.
